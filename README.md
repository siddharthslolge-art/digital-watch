# Digital Watch

A simple Digital Watch made using Python and Tkinter.

## Features

- Shows current time
- Updates every second
- Easy and clean interface
- Lightweight application

## Technologies Used

- Python
- Tkinter

## How to Run

1. Install Python
2. Download the project files
3. Open terminal in the project folder
4. Run:

```bash
python digital_watch.py
```
